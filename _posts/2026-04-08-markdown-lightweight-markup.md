## What is Markdown?

**Markdown** is a lightweight markup language created by John Gruber in 2004. It allows you to write formatted content using plain text that is easy to read and write, then convert it to HTML.

## Why Markdown?

### Advantages
- **Simple syntax** - Learn in 10 minutes
- **Human-readable** - Raw text is readable even without rendering
- **Platform-independent** - Works everywhere
- **Convertible** - Can be converted to HTML, PDF, DOCX, LaTeX, and more
- **Version-control friendly** - Works perfectly with Git

## Basic Markdown Syntax

| Element | Syntax |
|---------|--------|
| Heading 1 | `# Heading` |
| Heading 2 | `## Heading` |
| Bold | `**bold text**` |
| Italic | `*italic text*` |
| Link | `[title](https://example.com)` |
| Image | `![alt](image.jpg)` |
| Code | `` `code` `` |
| Code block | ```` ```code block``` ```` |
| Unordered list | `- item` |
| Ordered list | `1. item` |
| Table | `\| col1 \| col2 \|` |

## Markdown vs Other Formats

| Feature | Markdown | LaTeX | HTML |
|---------|----------|-------|------|
| Learning curve | Low | High | Medium |
| Readability | High | Low | Medium |
| Best for | Docs, blogs | Academic papers | Web pages |
